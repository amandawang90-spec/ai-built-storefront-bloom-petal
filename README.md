# 🌸 Bloom & Petal — Handmade Jewelry Shop

A fully frontend, single-file e-commerce shop for handcrafted jewelry. Built with vanilla HTML, CSS, and JavaScript — no frameworks, no dependencies.

🌐 **Live demo:** [bloom-and-netlify.netlify.app](https://bloom-and-netlify.netlify.app)

---

## ✨ Features

- 🛍️ **Product catalog** with categories (earrings, necklaces, rings, bracelets) and sale section
- 🔍 **Live search** with dropdown results
- 🛒 **Shopping cart** with quantity management
- 💳 **Checkout flow** with shipping calculation
- 🏆 **Loyalty points system** — earn and redeem points on orders
- 🌍 **Multilingual** — German, English, French, Dutch, Spanish
- 📄 **Invoice download** per order
- 📱 **Responsive design** — works on mobile and desktop
- 🎨 **Soft rose/blush aesthetic** with Playfair Display typography

---

## 🚀 Getting Started

No build step needed. Just open `index.html` in your browser — or deploy anywhere static.

```bash
git clone https://github.com/amandawang90-spec/ai-built-storefront
cd bloom-and-petal
open index.html
```
